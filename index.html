Наши навыки и с чем мы работаем:

HTML 5

Язык гипертекстовой разметки позволят создавать каркас сайта

CSS 3

Язык описания внешнего вида, позволяет украшать HTML разметку

JavaScript

Многонаправленый язык, в основном используеться для создания скриптов придающих динамичность странице

jQuery

JavaScript библиотека упрощающая реализацию некоторых скриптов

Sass

Язык в последствии превращающийся в CSS, создан для упрощения написания CSS кода

Photoshop

Графический редактор, в нашем случае используется для создания макетов и редактирования изображений

Illustrator

Редактор векторной графики, в основном мы его используем для создания иконок

InDesign

Программа в основном используемая для создания макетов

Figma

Графический редактор используемый нами для создания макетов

WebStrom

Среда разработки используемая нашей командой для написания HTML, CSS и JavaScript кода

VS Code

Среда разработки используемая нашей командой для написания HTML, CSS и JavaScript кода

Git

Система управления версиями, в нашем случае используется для работы с GitHub
